# ToxiGram (Social Media Toxic Neutralizer)

## 📌 Description
This project detects toxic comments and replaces them with neutral words using NLP.

## 🚀 Features
- Instagram-like UI
- Comment system
- Toxic word detection
- Automatic neutralization

## 🛠️ Tech Stack
- Python
- NLP (NLTK / sklearn)
- HTML, CSS, JS

## ▶️ How to Run
1. Clone repo
2. Install requirements
3. Run app
